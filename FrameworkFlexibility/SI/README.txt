An example input files of each type of simulation and the forcefield optimized configurations of each MOF are provide in this ZIP:
    1. DFT optimization (LUXFOA)
    2. Forcefield optimization (LUXFOA)
    3. MSD for fast diffusing cases (O2 in flexible and rigid LUXFOA)
    4. dcTST for slow diffusing cases (CCl4 in flexible ABAYIO)
    5. data files of the force field optimized MOFs (configurations used for diffusivity simulation)

"in.****" in each folder is the main input file of LAMMPS
"****.data" is the initial configuration file of each material
"****.colvars" are the auxiliary files for the collective variables package

Specific annotations are added in each file as comments, please check the software manuals for the detailed usage of commands.
    VASP Manual https://www.vasp.at/wiki/index.php/The_VASP_Manual
    LAMMPS Manual https://lammps.sandia.gov/doc/Manual.html .

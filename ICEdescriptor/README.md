# Interaction in Confined Environment (ICE): A Comprehensive Descriptor for Quantifying Guest-Host Interactions in MOFs

A physically interpretable descriptor for characterizing guest-host interactions in metal-organic frameworks (MOFs).

## Dependencies

- Python(>=3)

- NumPy （1.26.4）
